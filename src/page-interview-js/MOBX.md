#### mobx

- mobx 是一个库，通过函数响应式编程使状态管理变得简单和可扩展
  ![mobxstatus](/clipboard.png)
- Observable state (可观察的状态)
- Computed values (计算值)
- Reactions (反应)
  - React 组件
  - 自定义 reactions
    - MobX 会对在执行跟踪函数期间读取的任何现有的可观察属性做出反应。
- Actions (动作)
  - 状态应该以某种方式来更新
  - React 组件只是状态的华丽展示，而状态的衍生由 MobX 来管理
